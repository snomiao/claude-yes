Yes! Claude

Automated Claude CLI Wrapper Tool

What is Yes! Claude?

Yes! Claude is a wrapper tool that automates interactions with the Claude CLI by automatically handling common prompts and responses. It works exactly like the claude command but automatically responds "Yes" to confirmation prompts, allowing Claude to run continuously until your task is complete.

Installation

First, install Claude Code globally:

npm install -g @anthropic-ai/claude-code

Then install this project:

npm install claude-yes -g

Usage

claude-yes [command] [prompts] # works exactly same as `claude` command, and automatically says "Yes" to all yes/no prompts # e.g. claude-yes "run all tests and commit current changes"

The tool will:

  1. Run Claude Code
  2. Whenever Claude gets stuck on yes/no prompts, automatically say YES to proceed

How it Works

The tool uses node-pty to spawn and manage the Claude CLI process, looking for patterns like:

‚ùØ 1. Yes 2. No

When detected, it automatically sends a carriage return to select "Yes" and continue.
